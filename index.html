Getting the project running locally

  1. Checkout the code from github
  2. Install ruby
  3. Install the jekyll gem: gem install jekyll
  4. Navigate to your standards folder and use jekyll
  5. Visit http://localhost:4000/standards/

Helpers

There is a helper script that will checkout the gh-pages rebase your changes from the master branch and then re-checkout the master branch:


$ ./bin/rebase.sh
